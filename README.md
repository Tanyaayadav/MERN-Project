# 📚 BookStore Web Application

A full-stack MERN (MongoDB, Express.js, React, Node.js) bookstore web app that offers a seamless and responsive user experience with features like user authentication, theme toggling, interactive UI components, and conditional access to content.

---

## 🔥 Features

- **Login & Signup**: Secure user authentication and conditional content rendering.
- **Dynamic Book Cards**: Free books displayed to all users; paid books unlocked after signup.
- **Hover Animation**: Book cards scale on hover for an interactive feel.
- **Slider Functionality**: Highlighted books or categories shown via a slider on the homepage.
- **Dark/Light Mode**: Toggle between light and dark themes across the site.
- **Navigation Bar**: Includes Home, Courses, Contact, and About links for smooth navigation.

---

## 🛠️ Tech Stack

- **Frontend**: React.js, Tailwind CSS / CSS Modules  
- **Backend**: Node.js, Express.js  
- **Database**: MongoDB (with Mongoose)  
- **Authentication**: JWT & bcrypt  
- **Other**: React Router, Context API / Redux (if used), dotenv
